# Keyboard Shortcuts

All shortcuts are remappable from *Settings → Hotkeys*. The defaults are listed below.

> On macOS, every `Ctrl` below is `⌘` (Command).

## General

| Action | Default |
|--------|---------|
| Open Settings | `Ctrl+,` |
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Shift+Z` |
| Copy | `Ctrl+C` |
| Paste | `Ctrl+V` |
| Save | `Ctrl+S` |
| Select All | `Ctrl+A` |
| Play / Pause | `Space` |
| Delete Selection | `Delete` |

## Placement & view

| Action | Default |
|--------|---------|
| Create Star Power from Selection | `Ctrl+P` |
| Create Solo from Selection | `Ctrl+L` |
| Select Tool | `1` |
| Place Tool | `2` |
| Erase Tool | `3` |
| Zoom In | `Ctrl+=` |
| Zoom Out | `Ctrl+-` |
| Reset Zoom | `Ctrl+0` |

## Movement

| Action | Default |
|--------|---------|
| Nudge Left | `←` |
| Nudge Right | `→` |
| Nudge Up / Fret +1 | `↑` |
| Nudge Down / Fret −1 | `↓` |

## Modifiers

| Action | Default |
|--------|---------|
| Toggle Cymbal / Tap | `S` |
| Toggle Ghost / HOPO | `G` |
| Toggle Accent | `F` |
| Toggle Open / Kick | `O` |
| Toggle Star Power Mode | `P` |
| Toggle Solo Mode | `L` |
| Toggle Talkie Mode | `T` |

## Remapping

1. Open *Settings → Hotkeys*.
2. Click any binding to start recording, then press the new combination.
3. Conflicts are detected automatically — duplicates are highlighted in red.
4. Press *Reset to Defaults* in the modal footer to undo all changes.
